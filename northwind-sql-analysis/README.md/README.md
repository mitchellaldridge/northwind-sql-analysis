# Northwind SQL Business Analysis

This project explores business questions using the Northwind sample database.

## Database
Northwind (PostgreSQL sample database)

## Questions explored
- Which products generate the most revenue?
- Which countries generate the most sales?
- Who are the most valuable customers?
- How do sales change over time?
- Which employees drive the most revenue?

## Tools
- PostgreSQL
- DataGrip
- SQL
